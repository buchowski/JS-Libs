Class F.AjaxConnection

Use for all ajax-y stuff within simulations. Among other benefits, it re-tries connections on failure, does param parsing etc.

Constructor

Parameters:
url <String> url to connect to
settings <Object> Configuration options; see configs section below for values
ajaxOptions <Object> default jQuery ajax options

Configuration Attributes

icon - String

Path to icon for copy image
Default Value: "../img/page_white_paste.png"

onFailure - String

Failure handler on failed copy; Displays an alert/Dialog by default
Default Value: "ClipBoard.copyFailure"

onSuccess - String

Success handler on successful copy; Displays an alert/Dialog by default
Default Value: "ClipBoard.copySuccess"

tries - {Number}

The no. of times to retry request. Set to 1 for polling kinda stuff
Default Value: 2
